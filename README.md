A simple task management app built with React and Vite.

Features
  Create tasks
  Mark tasks as done
  Delete tasks

Tech Stack
  React
  Vite

Usage
  Clone the repo.
  Install dependencies: npm install.
  Run: npm run dev.
  Open in your browser.
